Ubuntu et Amazon sont deux systèmes d'exploitation populaire. Ubuntu est très connu dans le monde des informaticiens tandis que Amazon Linux est surtout
conçu pour le travail dans AWS et est disponible gratuitement dans les instances EC2.

Dans la suite de l'article, une comparaison détaillée sera faite des deux systèmes.

### Ubuntu

Ubuntu est une distribution linux qui est reconnu pour son interface facile à prendre en main même pour les débutants.
